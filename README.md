# install:
```bash
$ yarn install
```
or 
```bash
$ npm install
```


# debug:
### First to run json-server to run:
```bash
$ npm run json-server
```

### Then to run project:
```bash
$ yarn dev
```
or 
```bash
$ npm run dev
```
and check the result on: 
[http://localhost:3000](http://localhost:3000) 
with your browser to check the project


# build:
```bash
$ yarn build
```
or 
```bash
$ npm run build
```

npx create-next-app
